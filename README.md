# Análisis Estadístico sobre Hábitos Saludables en Jóvenes Universitarios

## Descripción del Proyecto
Este proyecto presenta un análisis de inferencia estadística sobre hábitos saludables en estudiantes universitarios. El objetivo principal es estudiar la relación entre el ejercicio físico, los hábitos de sueño, la alimentación y el nivel de estrés mediante técnicas estadísticas aplicadas en Python utilizando Jupyter Notebook.

El análisis incluye generación de datos simulados, exploración estadística, modelado de distribuciones, intervalos de confianza y pruebas de hipótesis para evaluar patrones de comportamiento en la población estudiantil.

---
## Objetivos

- Analizar la relación entre actividad física y horas de sueño.
- Evaluar cómo la alimentación puede influir en el nivel de estrés.
- Aplicar técnicas de inferencia estadística para estimar parámetros poblacionales.
- Realizar pruebas de hipótesis para identificar relaciones significativas entre variables.

--
# Variables Analizadas
El estudio considera las siguientes variables:

- Horas de sueño (variable continua)
- Días de actividad física por semana (variable discreta)
- Calidad de alimentación (variable ordinal)
- Nivel de estrés (variable ordinal)

El dataset utilizado contiene **200 registros simulados** que representan estudiantes universitarios.
---
## Metodología

El análisis se realizó siguiendo principios del método científico:

1. Planteamiento de una pregunta de investigación
2. Definición de hipótesis estadísticas
3. Simulación de un dataset representativo
4. Análisis exploratorio de datos
5. Aplicación de distribuciones de probabilidad
6. Uso del Teorema del Límite Central
7. Cálculo de intervalos de confianza
8. Realización de pruebas de hipótesis

## Resultados

Los resultados muestran evidencia estadística de que la actividad física tiene un impacto significativo en las horas de sueño de los estudiantes universitarios.

A través de pruebas de hipótesis (t-test) se encontró un valor p menor a 0.05, lo que permitió rechazar la hipótesis nula y concluir que existe una relación significativa entre ejercicio y patrones de sueño.

---

## Conclusiones
El análisis sugiere que mantener hábitos saludables, como realizar actividad física regularmente, puede contribuir a mejorar la calidad del sueño y el bienestar general de los estudiantes universitarios.

Además, este proyecto demuestra cómo las herramientas de inferencia estadística pueden utilizarse para analizar problemas reales relacionados con la salud y el comportamiento en poblaciones estudiantiles.
